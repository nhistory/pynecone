# pynecone

This is a project for INFT3075 group presentation

# requirements
## virtual environment
1. Install
```
python -m venv venv
```
2. Run virtual environment
```
venv\Scripts\activate.bat
```
## Install pynecone
```
pip install pynecone
```
## Create project
```
mkdir my_app_name
cd my_app_name
pc init
```
## Run project
```
pc run
```
